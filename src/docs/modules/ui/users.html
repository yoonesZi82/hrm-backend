Users (کاربران)

این ماژول مدیریت حساب‌های کاربری سیستم را بر عهده دارد.

وظایف

  • ایجاد کاربر
  • مشاهده لیست کاربران
  • ویرایش اطلاعات کاربر
  • حذف کاربر
  • تغییر نقش (Role)
  • فعال/غیرفعال کردن کاربر

نکات

  • هر User می‌تواند یک Employee داشته باشد
  • این ماژول فقط مربوط به اکانت است، نه اطلاعات شغلی
