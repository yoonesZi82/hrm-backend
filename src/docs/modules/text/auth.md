# 🔐 Auth (احراز هویت)

## توضیح

این ماژول مسئول مدیریت ورود کاربران و امنیت سیستم است.

## وظایف

- ثبت‌نام (Register)
- ورود (Login)
- تولید Access Token
- تولید Refresh Token
- مدیریت Logout
- مدیریت نقش‌ها (Role-based access)

## نکات

- از JWT استفاده می‌شود
- Refresh Token در Cookie ذخیره می‌شود
- Access Token در Header ارسال می‌شود
